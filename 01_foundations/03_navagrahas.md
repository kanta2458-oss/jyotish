## 1. グラハとは

**Graha（グラハ / 惑星）** とは、Sanskrit語で「つかむもの・支配するもの」を意味します。  
天体が人間の意識や運命に影響を与え、「把握（grip）する」という概念が語源です。

**Navagraha（ナヴァグラハ / 九惑星）** は *nava*（九）+ *graha*（惑星）で、Jyotishで使う9つの天体を指します。

> 西洋占星術の10天体（太陽〜冥王星）とは異なり、Jyotishでは**天王星・海王星・冥王星は使用しません**。  
> 代わりに影の惑星ラーフとケートゥが重視されます。

## 3. 各惑星の詳細

### ☉ Surya（スーリャ / 太陽）

| 属性 | 内容 |
|------|------|
| 性質 | 自然凶星（Naisargika Krura） |
| グナ | Sattva（サットヴァ） |
| 元素 | 火（Agni） |
| 支配星座 | 獅子座（Simha） |
| 高揚（Ucha） | 牡羊座（Mesha）10° |
| 減弱（Neecha） | 天秤座（Tula）10° |
| カラカ（表す領域） | 魂（Atman）、父、権威、政府、健康、自我 |
| 体の部位 | 心臓、骨、目（右） |
| 色 | 赤銅色 |

太陽はチャートの中で**ラーグナ（自己）の象徴**となる最重要惑星のひとつ。  
太陽の強さはその人の自信・リーダーシップ・健康を示します。

---

### ☽ Chandra（チャンドラ / 月）

| 属性 | 内容 |
|------|------|
| 性質 | 自然吉星（満月時）/ 自然凶星（新月前後） |
| グナ | Sattva（サットヴァ） |
| 元素 | 水（Jala） |
| 支配星座 | 蟹座（Karka） |
| 高揚（Ucha） | 牡牛座（Vrishabha）3° |
| 減弱（Neecha） | 蠍座（Vrishchika）3° |
| カラカ | 心、母、感情、直感、水、記憶 |
| 体の部位 | 心、血液、体液、左目 |

月はJyotishで**最も重要な惑星のひとつ**で、心（マナス）の象徴です。  
生まれた時の月の星座（**Rashi / ラーシ**）と月のナクシャトラはチャート解釈の出発点となります。

---

### ♂ Mangala（マンガラ / 火星）

| 属性 | 内容 |
|------|------|
| 性質 | 自然凶星 |
| グナ | Tamas（タマス） |
| 元素 | 火（Agni） |
| 支配星座 | 牡羊座（Mesha）、蠍座（Vrishchika） |
| 高揚（Ucha） | 山羊座（Makara）28° |
| 減弱（Neecha） | 蟹座（Karka）28° |
| カラカ | 行動力、兄弟、土地・不動産、争い、手術、軍隊 |
| 体の部位 | 血液、筋肉、骨髄、鼻 |

火星は**行動と意志**の惑星。強い火星はエネルギーと勇気を与えますが、  
凶影響が強まると怒り・争い・事故のテーマが出てきます。

---

### ☿ Budha（ブダ / 水星）

| 属性 | 内容 |
|------|------|
| 性質 | 中性（一緒にいる惑星の性質を取る） |
| グナ | Rajas（ラジャス） |
| 元素 | 地（Prithvi）/ 風（Vayu） |
| 支配星座 | 双子座（Mithuna）、乙女座（Kanya） |
| 高揚（Ucha） | 乙女座（Kanya）15° |
| 減弱（Neecha） | 魚座（Meena）15° |
| カラカ | 知性、言語、コミュニケーション、商業、数学、神経 |
| 体の部位 | 神経系、皮膚、声、肺 |

水星は**知性とコミュニケーション**の惑星。最も太陽に近く、常に太陽の近くに位置します。  
中性惑星なので、吉星と合すれば吉、凶星と合すれば凶の影響を受けやすいのが特徴です。

---

### ♃ Guru / Brihaspati（グル / ブリハスパティ / 木星）

| 属性 | 内容 |
|------|------|
| 性質 | 最大の自然吉星 |
| グナ | Sattva（サットヴァ） |
| 元素 | 空（Akasha） |
| 支配星座 | 射手座（Dhanu）、魚座（Meena） |
| 高揚（Ucha） | 蟹座（Karka）5° |
| 減弱（Neecha） | 山羊座（Makara）5° |
| カラカ | 知恵、精神性、師、子供、富、宗教、拡大 |
| 体の部位 | 肝臓、股関節、脂肪組織 |

木星は「グル（師）」とも呼ばれ、**最大の吉星**です。  
木星がアスペクト（視線）する惑星や室は保護・拡大の影響を受けます。

---

### ♀ Shukra（シュクラ / 金星）

| 属性 | 内容 |
|------|------|
| 性質 | 自然吉星 |
| グナ | Rajas（ラジャス） |
| 元素 | 水（Jala） |
| 支配星座 | 牡牛座（Vrishabha）、天秤座（Tula） |
| 高揚（Ucha） | 魚座（Meena）27° |
| 減弱（Neecha） | 乙女座（Kanya）27° |
| カラカ | 愛、美、芸術、贅沢、女性、快楽、結婚 |
| 体の部位 | 生殖器、腎臓、眼 |

金星は**愛と美**の惑星。物質的繁栄・芸術・人間関係を象徴します。  
特に女性のチャートや結婚のテーマで重要な役割を担います。

---

### ♄ Shani（シャニ / 土星）

| 属性 | 内容 |
|------|------|
| 性質 | 最大の自然凶星 |
| グナ | Tamas（タマス） |
| 元素 | 風（Vayu） |
| 支配星座 | 山羊座（Makara）、水瓶座（Kumbha） |
| 高揚（Ucha） | 天秤座（Tula）20° |
| 減弱（Neecha） | 牡羊座（Mesha）20° |
| カラカ | 試練、忍耐、老い、奉仕、隠者、業（カルマ） |
| 体の部位 | 骨、歯、膝、神経 |

土星は**カルマの惑星**。遅延・制限・試練を通じて成長と規律をもたらします。  
土星の影響が強い期間（ダシャー・サデサティ）は挑戦の時期となることが多いですが、  
誠実に取り組めば大きな成果をもたらす惑星でもあります。

## 4. 影の惑星 — Chaya Graha（チャーヤー・グラハ）

**Rahu（ラーフ）** と **Ketu（ケートゥ）** は、実際の天体ではなく**月の軌道と黄道が交差する点**です。  
（*chaya* = 影）

- **Rahu（ラーフ / 月の北交点）**: 執着・物質・欲望・幻想のエネルギー。増大・拡大の働き。
- **Ketu（ケートゥ / 月の南交点）**: 手放し・精神性・解脱・過去生のカルマ。分離・孤立の働き。

ラーフとケートゥは常に**正反対の星座**に位置し、180°離れています。  
（例：ラーフが牡羊座にいれば、ケートゥは天秤座）

| 属性 | Rahu（ラーフ） | Ketu（ケートゥ） |
|------|--------------|----------------|
| 象徴 | 頭（欲望） | 尾（解脱） |
| 性質 | 凶星（物質的執着） | 凶星（精神的切断） |
| カラカ | 外国、テクノロジー、毒、幻想 | 精神性、解脱、秘教、手術 |
| 支配星座 | 水瓶座（一説）/ Mithuna | 蠍座（一説）/ Dhanu |
| 高揚 | 双子座（Mithuna） / 牡牛座（諸説あり） | 射手座（Dhanu） / 蠍座 |

---

## 5. 吉星と凶星

### 自然吉星（Naisargika Shubha Graha）

チャート内の惑星配置に関わらず、本来的に吉の影響をもたらす惑星：

**木星（Guru）> 金星（Shukra）> 水銀（Budha） > 満月のチャンドラ**

### 自然凶星（Naisargika Krura Graha / Paapa Graha）

本来的に凶の影響をもたらす惑星：

**土星（Shani）> 火星（Mangala）> 太陽（Surya）> 欠けている月 > ラーフ > ケートゥ**

> **重要**: 「凶星」は「悪い星」という意味ではありません。  
> 凶星はより強く、鮮明なエネルギーを持つ星で、試練や変容をもたらします。  
> チャート内での配置・支配室・アスペクトによって、凶星も大きな良い結果をもたらします。

---

## 6. 惑星の自然的意味（Karaka / カラカ）

**Karaka（カラカ / 指示星）** とは、その惑星が自然に象徴・表示するライフテーマのことです。

| カラカ | 意味 | 主担当惑星 |
|--------|------|-----------|
| Atmakaraka（アートマカラカ） | 魂 | 太陽 |
| Amatyakaraka（アマートヤカラカ） | 職業・キャリア | 水星 |
| Bhrathrukaraka（ブラートリカラカ） | 兄弟 | 火星 |
| Matrukaraka（マートリカラカ） | 母 | 月 |
| Putrakaraka（プトラカラカ） | 子供 | 木星 |
| Gnatikaraka（ニャーティカラカ） | 競争・病気 | 土星 |
| Darakaraka（ダーラカラカ） | パートナー・配偶者 | 金星 |
