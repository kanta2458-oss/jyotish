## 1. バーヴァとは

**Bhava（バーヴァ / 室・ハウス）** は、出生チャートをラグナ（アセンダント）から数えた12の「生活領域」です。  
（*bhava* = 存在・状態・領域）

各バーヴァはそれぞれの**ライフテーマ**（健康・富・兄弟・家族など）を表しています。

### ラーシとバーヴァの違い（重要）

| | ラーシ（Rashi） | バーヴァ（Bhava） |
|-|---------------|----------------|
| 定義 | 黄道の30°区画（星座） | ラグナから数えたライフ領域 |
| 数 | 12 | 12 |
| 基準点 | 黄道の固定位置 | ラグナ（出生時の上昇点） |
| 意味 | 惑星の色・性質を変える | その惑星がどのテーマに関わるかを示す |

> Jyotishの全宮式（Whole Sign House System）では、ラグナの星座全体が第1室となり、  
> 以降の星座がそのまま第2室・第3室…となります。

## 3. バーヴァの分類

バーヴァは性質によっていくつかのグループに分類されます。この分類を理解することでチャート解釈が大きく変わります。

### Kendra（ケンドラ / 角宮）— 最も重要

**1、4、7、10室**

*kendra* = 中心・軸

チャートの四隅にあたる最も強力な室。ここにある惑星は強く表現されます。  
特に1室（ラグナ）と10室（カルマ）は最も重要とされます。

```
          10
        ┌────┐
    7   │    │   1
  ──────┤    ├──────
        │    │
        └────┘
           4
```

### Trikona（トリコナ / 三角宮）— 最も吉

**1、5、9室**

*tri* = 三、*kona* = 角

ダルマのハウスとも呼ばれる最も吉祥な室。特に5室と9室を「ラクシュミー・スターナ（Lakshmi Sthana / 女神の場所）」と言います。

> **Kendra + Trikona の組み合わせ**: 同じ惑星がケンドラとトリコナ両方の支配星となる場合、**Raja Yoga（ラジャ・ヨーガ / 王の組み合わせ）** となり、強い成功・地位をもたらします。

### Dusthana（ドゥスターナ / 凶室）

**6、8、12室**

*dus* = 難しい、*sthana* = 場所

試練・困難・喪失に関わる室。ここに惑星が多いチャートは人生の挑戦が多い傾向があります。  
ただし、6室は競争力・奉仕の精神、8室は深い洞察力・神秘、12室は精神性・解脱の場でもあります。

> **Vipareeta Raja Yoga（ヴィパリータ・ラジャ・ヨーガ）**: 凶室（6、8、12室）の支配星がお互いに関係し合う場合、逆説的に良い結果をもたらす特殊ヨーガです。

### Upachaya（ウパチャーヤ / 成長室）

**3、6、10、11室**

*upachaya* = 増大・成長

時間とともに改善・成長する室。ここに凶星があると、最初は困難でも後から好転することが多いです。  
（特に土星・火星はウパチャーヤ室で力を発揮します）

### Maraka（マラカ / 死を示す室）

**2、7室**

これらの室の支配星は**マラカ**と呼ばれ、特定のダシャー期間に死・重病・大きな喪失を示すことがあります。

## 4. バーヴァの支配星（Bhava Lord / バーヴァ・ロード）

各バーヴァには支配惑星（**Bhava Lord / バーヴァ・ロード**）が存在します。  
バーヴァ・ロードはそのバーヴァのテーマを「担当する」惑星であり、  
その惑星の位置・状態・アスペクトがそのバーヴァのテーマの展開を示します。

例えば：
- **1室支配星**（ラグナ・ロード）= その人の人生の方向性・健康の主要指示星
- **10室支配星**（カルメーシャ）= 職業・キャリアの方向性を示す
- **7室支配星** = 結婚・パートナーシップの質を示す

バーヴァ・ロードの支配星は、ラーシ（星座）によって決まります（前のノート参照）。

## 5. チャートでの位置確認

南インド式チャート（グリッド形式、時計回りに星座が固定）と  
北インド式チャート（ダイヤモンド形式、数字でハウスを示す）があります。

**南インド式（South Indian Chart）**:
```
┌────┬────┬────┬────┐
│ Pi │ Ar │ Ta │ Ge │
├────┼────┼────┼────┤
│ Aq │    │    │ Ca │
├────┼────┼────┼────┤
│ Cp │    │    │ Le │
├────┼────┼────┼────┤
│ Sa │ Sc │ Li │ Vi │
└────┴────┴────┴────┘
（星座が固定で配置。ラグナ星座が1室となる）
```

**北インド式（North Indian Chart）**:
```
    ┌───┬───┐
    │ 12│ 1 │
  ┌─┤   ├───┤─┐
  │11│   │ 2 │
  ├─┤   ├───┤─┤
  │10│   │ 3 │
  └─┤   ├───┤─┘
    │ 9 │ 4 │
    └───┴───┘
（ハウス番号が固定で配置）
```
