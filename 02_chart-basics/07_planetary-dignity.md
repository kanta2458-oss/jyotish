# 惑星の品位 — Planetary Dignity（グラハの強さ）

> **難易度**: 初級〜中級  
> **前提ファイル**: [03_navagrahas.md](../01_foundations/03_navagrahas.md), [04_rashis.md](../01_foundations/04_rashis.md)  
> **キーワード**: Ucha, Neecha, Svagraha, Moolatrikona, Shadbala, Neecha Bhanga

---

## 1. 惑星の品位とは

**惑星の品位（Planetary Dignity）** とは、ある惑星が特定の星座にいるときの「力の強さ・表現の質」を示す概念です。

同じ惑星でも、どの星座にいるかによってその力は大きく変わります。  
これを理解することで「その惑星が力強くテーマを表現できるか」「弱まって課題となるか」が分かります。

---

## 2. 五段階の品位

### 1. Ucha（ウーチャ / 高揚）— 最高の力

**Exaltation（エグザルテーション）** とも呼ばれます。  
惑星が最も力強く輝ける星座です。その惑星の本来の性質が最大限に発揮されます。

| 惑星 | 高揚星座 | 高揚度数 |
|------|---------|---------|
| 太陽（Surya） | 牡羊座（Mesha） | 10° |
| 月（Chandra） | 牡牛座（Vrishabha） | 3° |
| 火星（Mangala） | 山羊座（Makara） | 28° |
| 水星（Budha） | 乙女座（Kanya） | 15° |
| 木星（Guru） | 蟹座（Karka） | 5° |
| 金星（Shukra） | 魚座（Meena） | 27° |
| 土星（Shani） | 天秤座（Tula） | 20° |
| ラーフ（Rahu） | 双子座（Mithuna）または牡牛座（諸説あり） | — |
| ケートゥ（Ketu） | 射手座（Dhanu）または蠍座（諸説あり） | — |

> 高揚度数（**Paramocchabindu / パラモッチャビンドゥ**）に正確に位置するとき、その惑星の力は最大となります。

---

### 2. Moolatrikona（ムーラトリコナ）— 非常に強い

Ucha（高揚）の次に強い状態。自宮よりやや強く、高揚より少し弱い位置づけです。  
（*moola* = 根、*trikona* = 三角形 → 「根源的な三角形」）

| 惑星 | ムーラトリコナ星座 | 範囲 |
|------|----------------|------|
| 太陽（Surya） | 獅子座（Simha） | 0°〜20° |
| 月（Chandra） | 牡牛座（Vrishabha） | 4°〜30° |
| 火星（Mangala） | 牡羊座（Mesha） | 0°〜12° |
| 水星（Budha） | 乙女座（Kanya） | 16°〜20° |
| 木星（Guru） | 射手座（Dhanu） | 0°〜10° |
| 金星（Shukra） | 天秤座（Tula） | 0°〜15° |
| 土星（Shani） | 水瓶座（Kumbha） | 0°〜20° |

---

### 3. Svagraha / Svarashi（スヴァグラハ / 自宮）— 強い

惑星が自分の支配する星座にいる状態。「我が家にいる」ようなもので、安定して力を発揮します。

| 惑星 | 支配星座（自宮） |
|------|--------------|
| 太陽（Surya） | 獅子座（Simha） |
| 月（Chandra） | 蟹座（Karka） |
| 火星（Mangala） | 牡羊座（Mesha）、蠍座（Vrishchika） |
| 水星（Budha） | 双子座（Mithuna）、乙女座（Kanya） |
| 木星（Guru） | 射手座（Dhanu）、魚座（Meena） |
| 金星（Shukra） | 牡牛座（Vrishabha）、天秤座（Tula） |
| 土星（Shani） | 山羊座（Makara）、水瓶座（Kumbha） |

> ラーフ・ケートゥは自宮の概念については諸説あります（流派により異なる）。

---

### 4. Neutral（中性）— 普通の力

友好的な惑星の星座にいる状態（**Mitra Rashi / ミトラ・ラーシ**）。  
特別に強くも弱くもなく、普通の力で機能します。

---

### 5. Neecha（ネーチャ / 減弱）— 最も弱い

**Debilitation（デビリテーション）** とも呼ばれます。  
高揚の正反対の星座（高揚星座から180°）にあたり、その惑星が最も力を発揮しにくい状態です。

| 惑星 | 減弱星座 | 減弱度数 |
|------|---------|---------|
| 太陽（Surya） | 天秤座（Tula） | 10° |
| 月（Chandra） | 蠍座（Vrishchika） | 3° |
| 火星（Mangala） | 蟹座（Karka） | 28° |
| 水星（Budha） | 魚座（Meena） | 15° |
| 木星（Guru） | 山羊座（Makara） | 5° |
| 金星（Shukra） | 乙女座（Kanya） | 27° |
| 土星（Shani） | 牡羊座（Mesha） | 20° |

---

## 3. 品位の比較まとめ

```
Ucha（高揚）> Moolatrikona（ムーラトリコナ）> Svagraha（自宮）> Neutral（中性）> Neecha（減弱）
   最強                                                                              最弱
```

---

## 4. 品位の解釈

### 高揚（Ucha）の解釈

高揚星座の惑星はその惑星のテーマを非常に強く、純粋に表現します。  
ただし「過剰」になりやすい面もあります。

例：**蟹座の木星（Ucha）** → 教育・精神性・子供・知恵のテーマが強く発展する。  
ただし過度な楽観主義・拡大しすぎる傾向も出る場合があります。

### 減弱（Neecha）の解釈

減弱は「弱い・機能しにくい」状態ですが、**必ずしも悪い結果をもたらすわけではありません**。

- 減弱惑星はその惑星のテーマで苦労・迷い・試練を経験しやすい
- 一方で、その苦労を通じて深い洞察や成熟を得ることも多い
- 減弱の状態が解消される条件（**Neecha Bhanga**）がある

---

## 5. Neecha Bhanga（ネーチャ・バンガ / 減弱解除）

**Neecha Bhanga（ネーチャ・バンガ）** とは、減弱惑星の弱さが「解除・キャンセル」される条件です。  
（*neecha* = 減弱、*bhanga* = 破壊・解除）

Neecha Bhanガが成立すると、その惑星は減弱の試練を乗り越え、むしろ大きな力をもたらすとされます。  
（特に**Neecha Bhanga Raja Yoga**と呼ばれる場合、王のような成功をもたらします）

### 主なNeecha Bhanga の条件

1. **減弱惑星の支配星がケンドラ（1,4,7,10室）にいる**
2. **その惑星がケンドラにいる**
3. **その惑星を高揚させる惑星がケンドラにいる**
4. **減弱惑星の星座の支配星と高揚させる惑星がケンドラにいる**
5. **ナヴァムシャ（D-9）では高揚または自宮にいる**

例：**山羊座（Makara）の木星（Neecha）** の場合
- 山羊座の支配星は土星（Shani）
- 土星がケンドラにいれば → Neecha Bhanガ成立

---

## 6. Shadbala（シャドバラ / 六種の強さ）

**Shadbala（シャドバラ）** とは、惑星の総合的な強さを6つの観点から数値計算するシステムです。  
（*shad* = 六、*bala* = 力）

詳細な計算は複雑ですが、6つの要素は：

| # | 名称 | 内容 |
|---|------|------|
| 1 | Sthana Bala（スターナ・バラ） | 星座位置による強さ（高揚・自宮など） |
| 2 | Dig Bala（ディグ・バラ） | 方角による強さ |
| 3 | Kala Bala（カーラ・バラ） | 時間（昼/夜・月相など）による強さ |
| 4 | Cheshta Bala（チェシュター・バラ） | 惑星の動き（逆行・順行）による強さ |
| 5 | Naisargika Bala（ナイサルギカ・バラ） | 自然的な強さ（太陽が最強） |
| 6 | Drik Bala（ドリク・バラ） | アスペクトによる強さ |

初学者の段階では品位（高揚・自宮・減弱）と逆行（Vakri）の確認を優先しましょう。

---

## 7. 逆行（Vakri / ヴァクリ）

**Vakri（ヴァクリ / 逆行）** は惑星が見かけ上逆向きに動く現象です。  
逆行惑星はその惑星のテーマを「内向き」に処理する傾向があります。

- 太陽・月・ラーフ・ケートゥは逆行しません
- 逆行中の惑星はシャドバラで高くなる（Cheshta Bala）
- ただし表現は「屈折した」ものになりやすい
