# アスペクト — Drishti（ドリシュティ / 惑星の視線）

> **難易度**: 初級〜中級  
> **前提ファイル**: [03_navagrahas.md](../01_foundations/03_navagrahas.md), [05_bhavas.md](../01_foundations/05_bhavas.md)  
> **キーワード**: Drishti, Graha Drishti, Rashi Drishti, Full Aspect, Special Aspect

---

## 1. ドリシュティとは

**Drishti（ドリシュティ / 視線・アスペクト）** とは、惑星が自分の位置から「視線を向ける（見る）」方向のことです。  
（*drishti* = 視線・見ること）

Jyotishのアスペクトは**星座を単位**にして計算されます。  
これは西洋占星術の度数ベースのアスペクト（60°、90°、120°など）とは異なります。

> **西洋占星術との違い**:  
> 西洋 → 度数で計算（コンジャンクション、スクエア、トライン等）  
> Jyotish → 星座（バーヴァ）番号で計算

---

## 2. 全惑星共通：第7室アスペクト

**すべての惑星**は、自分がいる星座から数えて**第7番目の星座**を「完全に見る（Full Aspect）」のが基本です。

例：惑星が**蟹座（第4室）**にいる場合
- 第7室は蟹座から7番目 → **山羊座（第10室）**
- この惑星は山羊座（第10室）に完全なアスペクトを投げる

```
惑星の位置: X室
全惑星の基本アスペクト: X + 6 室（= 反対側の室）
```

---

## 3. 特殊アスペクト（グラハ・ドリシュティ）

基本の第7室アスペクトに加え、特定の惑星は**追加の特殊アスペクト**を持ちます。

### 火星（Mangala）の特殊アスペクト

自分の位置から **第4室** と **第8室** にも完全なアスペクトを投げます。

```
火星のアスペクト: 第4室、第7室、第8室
```

例：火星が**牡羊座（第1室）**にいる場合
- 第4アスペクト → 蟹座（第4室）
- 第7アスペクト → 天秤座（第7室）
- 第8アスペクト → 蠍座（第8室）

**解釈**: 火星のアスペクトはエネルギー・闘争心・手術のテーマをその室に注ぎます。  
強い火星のアスペクトはその室のテーマに強い活力と時に争いをもたらします。

---

### 木星（Guru）の特殊アスペクト

自分の位置から **第5室** と **第9室** にも完全なアスペクトを投げます。

```
木星のアスペクト: 第5室、第7室、第9室
```

例：木星が**蟹座（第4室）**にいる場合（高揚）
- 第5アスペクト → 蠍座（第8室）
- 第7アスペクト → 山羊座（第10室）
- 第9アスペクト → 魚座（第12室）

**解釈**: 木星のアスペクトは最大の吉星として、アスペクトした室を保護し拡大します。  
特に5・9室（トリコナ）へのアスペクトは精神的・物質的繁栄をもたらします。

---

### 土星（Shani）の特殊アスペクト

自分の位置から **第3室** と **第10室** にも完全なアスペクトを投げます。

```
土星のアスペクト: 第3室、第7室、第10室
```

例：土星が**天秤座（第7室）**にいる場合（高揚）
- 第3アスペクト → 射手座（第9室）
- 第7アスペクト → 牡羊座（第1室）
- 第10アスペクト → 蟹座（第4室）

**解釈**: 土星のアスペクトはその室のテーマに遅延・試練・規律・真剣さをもたらします。  
ラグナ（第1室）への土星アスペクトは外見や健康に土星的な性質（痩身・持久力・真面目さ）を与えます。

## 5. ラーフ・ケートゥのアスペクト

ラーフ（Rahu）とケートゥ（Ketu）のアスペクトについては**流派によって異なります**。

**一般的な見解**:
- ラーフは第5室・第7室・第9室にアスペクト（木星と同様とする流派）
- ケートゥは第1室・第5室・第9室にアスペクト
- ただし「ラーフ・ケートゥはアスペクトしない」とする流派も存在

初学者は「**ラーフ・ケートゥは在室による影響が主**」と理解しておくとよいでしょう。

---

## 6. アスペクトの強度

一部の流派では、アスペクトの強度に段階があるとされます：

| アスペクト | 強度 |
|-----------|------|
| 第7室（全惑星） | 完全（Full / 100%） |
| 火星の第4・8室 | 完全（Full / 100%） |
| 木星の第5・9室 | 完全（Full / 100%） |
| 土星の第3・10室 | 完全（Full / 100%） |

BPHS（根本聖典）では、特殊アスペクトは列挙された室への完全アスペクトとして扱います。

## 7. Rashi Drishti（ラーシ・ドリシュティ）

**Rashi Drishti（ラーシ・ドリシュティ / 星座アスペクト）** は、惑星ではなく**星座（ラーシ）自体**が見るアスペクトです。  
Jaimini（ジャイミニ）占星術で特に重視される概念です。

| 様式 | ドリシュティ（視線を向ける先） |
|------|---------------------------|
| Chara（活動宮：牡羊・蟹・天秤・山羊） | 3つの不動宮（ただし隣を除く）を見る |
| Sthira（不動宮：牡牛・獅子・蠍・水瓶） | 3つの活動宮（ただし隣を除く）を見る |
| Dvisvabhava（柔軟宮：双子・乙女・射手・魚） | お互いに見る |

初学者の段階では**Graha Drishti（グラハ・ドリシュティ / 惑星アスペクト）**を優先して理解しましょう。

## 8. アスペクトの解釈ポイント

### 吉星のアスペクト

**木星・金星・水星・満月**などの吉星がアスペクトする室は：
- その室のテーマが保護・強化される
- 良い結果がもたらされやすい
- 困難があっても乗り越えやすい

例：**木星が第7室（結婚の室）に第7アスペクト** → 良縁・誠実なパートナーを示す

### 凶星のアスペクト

**土星・火星・太陽・ラーフ・ケートゥ**などがアスペクトする室は：
- その室のテーマに試練・遅延・変容をもたらす
- 完全に否定的ではなく、厳しさや精錬の意味も

例：**土星が第4室（母・家・心の安定）をアスペクト** → 家庭生活での試練・母との距離感・精神的な修行

### 在室と支配星とアスペクトの組み合わせ

惑星の影響は：
1. **在室（Occupation）**: その惑星がいる室を最も直接的に影響
2. **アスペクト（Drishti）**: 視線を向けた室に影響
3. **支配（Lordship）**: 支配する室のテーマを担当

この3つを組み合わせることで、惑星のチャート内での役割が立体的に見えてきます。
