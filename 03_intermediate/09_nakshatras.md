## 1. ナクシャトラとは

**Nakshatra（ナクシャトラ / 二十七宿 / 月の宿）** は、黄道を**27等分**（各13°20'）した区画です。  
（*naksha* = 地図・星、*tra* = 宿）

月がほぼ1日に1つのナクシャトラを通過することから「月の宿」とも呼ばれます。  
（月の公転周期 ≒ 27.3日）

> **ラーシとナクシャトラの違い**  
> ラーシ（12星座）= 黄道を30°ずつ12分割  
> ナクシャトラ（27宿）= 黄道を13°20'ずつ27分割  
> 各ナクシャトラは各ラーシにまたがって存在します（1ラーシに約2.25ナクシャトラ）

ナクシャトラの体系はラーシ（星座）よりも古い歴史を持ち、ヴェーダ時代から使われてきました。

---

## 2. ナクシャトラの重要性

- **ヴィムショッタリー・ダシャー**: 月のナクシャトラからダシャーの開始惑星と残年数を計算
- **性格・気質**: 月のナクシャトラ（ジャンマ・ナクシャトラ）は月の星座より細かい気質を示す
- **相性診断（Kuta Matching）**: 結婚相性の判断にナクシャトラを使う
- **ムフールタ（吉日選び）**: 各ナクシャトラの性質を考慮して吉日を選定

## 4. ジャンマ・ナクシャトラ（Janma Nakshatra）

**Janma Nakshatra（ジャンマ・ナクシャトラ / 出生のナクシャトラ）** とは、  
出生時に月がいたナクシャトラのことです。

これはインドの伝統では「その人のナクシャトラ」として非常に重視されます。  
名前の選定・儀礼のタイミングなどもジャンマ・ナクシャトラを基準に行われます。

---

## 5. パダ（Pada / 四分割）

各ナクシャトラは4つの **Pada（パダ / 足 / 四半）** に分割されます。  
（*pada* = 足・四分の一）

各パダは3°20'で、全27ナクシャトラ × 4パダ = **108パダ**。  
この108はヒンドゥー哲学の聖数でもあります。

### パダとナヴァムシャの対応

各パダは12のラーシ（星座）に対応します。この対応が **Navamsha（ナヴァムシャ / D-9）** チャートの基礎となります。

| ナクシャトラ | 第1パダ | 第2パダ | 第3パダ | 第4パダ |
|------------|--------|--------|--------|--------|
| Ashwini | 牡羊 | 牡牛 | 双子 | 蟹 |
| Bharani | 獅子 | 乙女 | 天秤 | 蠍 |
| Krittika | 射手 | 山羊 | 水瓶 | 魚 |
| （以降繰り返し） | | | | |

## 6. ナクシャトラの三分類

ナクシャトラは性質によって三つのグループに分類されます。

### Deva（デーヴァ / 神聖なもの）
精神性・慈悲・高い目的を象徴するナクシャトラ

Ashwini, Mrigashira, Punarvasu, Pushya, Hasta, Swati, Anuradha, Shravana, Revati

### Manushya（マーヌシャ / 人間的なもの）
物質的・人間的な欲求と精神性のバランスを象徴

Bharani, Rohini, Ardra, Purva Phalguni, Uttara Phalguni, Purva Ashadha, Uttara Ashadha

### Rakshasa（ラークシャサ / 悪魔的なもの）
強烈なエネルギー・破壊と創造を象徴（必ずしも「悪い」わけではない）

Krittika, Ashlesha, Magha, Chitra, Vishakha, Jyeshtha, Mula, Dhanishtha, Shatabhisha, Purva Bhadrapada, Uttara Bhadrapada

## 7. 各ナクシャトラの性質（主要なもの）

### Ashwini（アシュヴィニー）
- **支配惑星**: ケートゥ　**主神**: アシュヴィン双神（医療神）
- **性質**: 速さ、治癒、開始、清潔感、鋭さ
- 何事も速く始め、素早く動く。医療・スポーツとの縁。

### Rohini（ローヒニー）
- **支配惑星**: 月　**主神**: ブラフマー（創造神）
- **性質**: 美、豊かさ、創造性、魅力、物質的繁栄
- 月のもっとも好む宿。美しさと豊かさのエネルギー。

### Ardra（アールドラー）
- **支配惑星**: ラーフ　**主神**: ルドラ（嵐・変容の神）
- **性質**: 嵐・変容・感情の激しさ・洞察力
- 破壊と再生のエネルギー。深い変容を経験する。

### Pushya（プシュヤ）
- **支配惑星**: 土星　**主神**: ブリハスパティ（木星）
- **性質**: 栄養・育む・保護・発展・最も吉祥とされる宿
- 月がPushyaにある時は特に吉祥（Pushya Nakshatra Yoga）。

### Magha（マガ）
- **支配惑星**: ケートゥ　**主神**: ピトリ（祖先の霊）
- **性質**: 権威・王室・先祖との繋がり・過去生のカルマ
- 先祖や系譜との深い縁を持つ。権力と威厳のエネルギー。

### Chitra（チトラ）
- **支配惑星**: 火星　**主神**: トヴァシュトリ（工匠神）
- **性質**: 輝き・芸術・創造性・完璧主義
- スパイカ（乙女座の一等星）に対応。美と技巧のエネルギー。

### Mula（ムーラ）
- **支配惑星**: ケートゥ　**主神**: ニルリティ（破壊の女神）
- **性質**: 根を断ち切る・深掘り・探求・破壊と再生
- 物事の根本を揺るがすような変化が起きやすいナクシャトラ。

## 8. ナクシャトラとヴィムショッタリー・ダシャーの関係

27ナクシャトラのそれぞれには**支配惑星**（Nakshatra Lord）が割り当てられています。  
この支配惑星が**ヴィムショッタリー・ダシャー**の計算の基礎となります。

ダシャーの順序は以下のように、ケートゥから始まり9惑星が繰り返します：

```
ケートゥ → 金星 → 太陽 → 月 → 火星 → ラーフ → 木星 → 土星 → 水星
（→ ケートゥ → 金星 → ...繰り返し）
```

月がどのナクシャトラにいるかで「どの惑星のダシャーから始まるか」と「残り何年か」が決まります。  
詳しくは次のノートで解説します。
