# ヴィムショッタリー・ダシャー — Vimshottari Dasha（惑星期間システム）

> **難易度**: 中級  
> **前提ファイル**: [09_nakshatras.md](./09_nakshatras.md)  
> **キーワード**: Dasha, Vimshottari, Antardasha, Pratyantardasha, Nakshatra Lord

---

## 1. ダシャーとは

**Dasha（ダシャー / 惑星期間）** とは、特定の惑星が人生の一定期間を「支配する」という概念です。  
（*dasha* = 状態・条件・期間）

Jyotishにはいくつかのダシャーシステムがありますが、最も広く使われるのが **Vimshottari Dasha（ヴィムショッタリー・ダシャー）** です。  
（*vimshottari* = 120 → 合計120年のサイクル）

ダシャー期間中は、その惑星のカラカ（象徴するテーマ）・ラーシ（星座）・バーヴァ（室）の  
テーマが人生の前面に出てきやすくなります。

## 2. ヴィムショッタリー・ダシャーの構造

### ダシャーの順序と年数

| 惑星 | Sanskrit | 年数 |
|------|----------|------|
| ケートゥ | Ketu | **7年** |
| 金星 | Venus（Shukra） | **20年** |
| 太陽 | Sun（Surya） | **6年** |
| 月 | Moon（Chandra） | **10年** |
| 火星 | Mars（Mangala） | **7年** |
| ラーフ | Rahu | **18年** |
| 木星 | Jupiter（Guru） | **16年** |
| 土星 | Saturn（Shani） | **19年** |
| 水星 | Mercury（Budha） | **17年** |
| **合計** | | **120年** |

この順序は固定で、120年で一周します。

### ダシャーの記憶法

```
ケ（7）金（20）太（6）月（10）火（7）ラ（18）木（16）土（19）水（17）
= 7+20+6+10+7+18+16+19+17 = 120年
```

## 3. スタートのダシャーを決める方法

出生時にどのダシャーから始まるかは、**月のナクシャトラ（ジャンマ・ナクシャトラ）**によって決まります。

### ナクシャトラとダシャー支配惑星の対応

各ナクシャトラには支配惑星が割り当てられています（9惑星 × 3回 = 27ナクシャトラ）：

| ダシャー惑星 | 支配するナクシャトラ（番号） |
|------------|------------------------|
| ケートゥ（Ketu） | 1（Ashwini）、10（Magha）、19（Mula） |
| 金星（Shukra） | 2（Bharani）、11（Purva Phalguni）、20（Purva Ashadha） |
| 太陽（Surya） | 3（Krittika）、12（Uttara Phalguni）、21（Uttara Ashadha） |
| 月（Chandra） | 4（Rohini）、13（Hasta）、22（Shravana） |
| 火星（Mangala） | 5（Mrigashira）、14（Chitra）、23（Dhanishtha） |
| ラーフ（Rahu） | 6（Ardra）、15（Swati）、24（Shatabhisha） |
| 木星（Guru） | 7（Punarvasu）、16（Vishakha）、25（Purva Bhadrapada） |
| 土星（Shani） | 8（Pushya）、17（Anuradha）、26（Uttara Bhadrapada） |
| 水星（Budha） | 9（Ashlesha）、18（Jyeshtha）、27（Revati） |

## 4. ダシャーの残余（Balance）を計算する

出生時、月がナクシャトラの途中にいる場合、そのナクシャトラの支配惑星のダシャーがまだ  
「途中から始まる」ことになります。

### 計算方法

```
ダシャー残余 = (ナクシャトラの残り度数 ÷ ナクシャトラ全体の度数) × そのダシャーの年数
```

各ナクシャトラの全体度数 = **13°20'**

**例：月がPushya（プシュヤ）の6°40'にある場合**
- Pushyaの開始：蟹座3°20'　終了：蟹座16°40'
- Pushya全体（13°20' = 13.33°）のうち、月は先頭から3°20'（3.33°）進んでいる
- 残り度数：13.33° − 3.33° = 10°
- 残り割合：10 ÷ 13.33 ≒ 0.75
- 土星ダシャー（19年）の0.75 → **残り約14年3ヶ月**

## 5. アンタルダシャー（Antardasha / 副期間）

各メインダシャー（**Mahadasha / マハーダシャー**）はさらに9つの副期間に分割されます。  
これを **Antardasha（アンタルダシャー / 副期間）** または **Bhukti（ブクティ）** と呼びます。

アンタルダシャーの順序もマハーダシャーと同じ惑星順で、  
**マハーダシャーの惑星からスタート**します。

例：**木星のマハーダシャー（16年）**のアンタルダシャー順序：

| アンタルダシャー惑星 | 期間 |
|------------------|------|
| 木星/木星 | 2年1ヶ月18日 |
| 木星/土星 | 2年6ヶ月12日 |
| 木星/水星 | 2年3ヶ月6日 |
| 木星/ケートゥ | 0年11ヶ月6日 |
| 木星/金星 | 2年8ヶ月0日 |
| 木星/太陽 | 0年9ヶ月18日 |
| 木星/月 | 1年4ヶ月0日 |
| 木星/火星 | 0年11ヶ月6日 |
| 木星/ラーフ | 2年4ヶ月24日 |
| **合計** | **16年** |

### プラティヤンタルダシャー（Pratyantardasha / 小副期間）

アンタルダシャーはさらに9分割されます。これを **Pratyantardasha（プラティヤンタルダシャー）** と呼びます。

> **実践では**: マハーダシャー → アンタルダシャーまでを主に使います。  
> プラティヤンタルダシャー以下は特定のイベントの精密な予測に使います。

## 6. ダシャーの解釈

### マハーダシャーの解釈

そのダシャー惑星が：

1. **どの星座にいるか**（ラーシ） → 惑星の色・性質
2. **どの室にいるか**（バーヴァ） → 影響するライフテーマ
3. **どのナクシャトラにいるか** → より細かい性質
4. **品位は何か**（高揚・自宮・減弱など） → 強さの程度
5. **何室の支配星か**（ロードシップ） → 担当するテーマ
6. **何がアスペクトしているか** → 外部からの影響

### 各惑星ダシャーの概略

| 惑星 | ダシャーのテーマ |
|------|--------------|
| ケートゥ（7年） | 霊的探求・過去清算・手放し・孤独・隠遁 |
| 金星（20年） | 愛・美・物質的繁栄・創造・芸術・結婚 |
| 太陽（6年） | 自己主張・権威・健康・父・政府・魂の目的 |
| 月（10年） | 感情・母・変化・直感・人々との関わり |
| 火星（7年） | 行動・競争・不動産・エネルギー・争い |
| ラーフ（18年） | 野心・執着・外国・物質的欲望・突然の変化 |
| 木星（16年） | 知恵・精神的成長・子供・師との出会い・繁栄 |
| 土星（19年） | 試練・忍耐・カルマ清算・職業的責任・老い |
| 水星（17年） | 知性・コミュニケーション・学習・ビジネス |

## 7. アンタルダシャーの組み合わせ解釈

マハーダシャー惑星とアンタルダシャー惑星の**関係性**が重要です。

### 惑星間の友好・敵対関係

| 惑星 | 自然的友人 | 自然的敵 |
|------|----------|---------|
| 太陽 | 月・火星・木星 | 土星・金星 |
| 月 | 太陽・水星 | なし |
| 火星 | 太陽・月・木星 | 水星・金星 |
| 水星 | 太陽・金星 | 月 |
| 木星 | 太陽・月・火星 | 水星・金星・土星 |
| 金星 | 水星・土星 | 太陽・月 |
| 土星 | 水星・金星 | 太陽・月・火星 |

友好的な惑星の組み合わせ（例：木星マハーダシャー / 月アンタルダシャー）では  
比較的スムーズな結果をもたらし、敵対的な組み合わせでは緊張や課題が生じやすくなります。

## 8. 実践例

### 例：月がArdra（アールドラー）の5°にある場合

1. **ナクシャトラ確認**: Ardra（双子座6°40'〜20°）の支配惑星 → **ラーフ**
2. **ダシャーの残余計算**:
   - Ardra全体13°20'のうち、月は先頭から(5° − 6°40' + 13°20' = 11°20' ... 実際の計算は)
   - 実際にはJyotish計算ソフトが自動計算します
3. **ダシャーの順序**: ラーフ(18年) → 木星(16年) → 土星(19年) → ...

### ダシャー期間の人生への影響

ダシャーは「その時期の主役となる惑星」と考えると理解しやすいです。

- **ラーフ・ダシャー（18年）**: 物質的・社会的な野心が前面に出る。外国・新テクノロジーとの縁。幻想からの目覚め。
- **木星・ダシャー（16年）**: 精神的成長・結婚・子供・教育・豊かさのテーマが展開する。
- **土星・ダシャー（19年）**: 最も試練が多い時期とされることが多いが、誠実に取り組めば大きな成果をもたらす。

## 9. ゴーチャラ（Gochar / トランジット）との組み合わせ

**Gochar（ゴーチャラ / トランジット）** とは現在の惑星位置がチャートに与える影響です。  
ダシャーとゴーチャラを組み合わせることで、より精密な時期の予測が可能になります。

一般的なルール：
- 木星のゴーチャルはラグナや月から数えた吉室への通過で良い結果をもたらす
- 土星のゴーチャルは特に月から数えた12・1・2室への通過（**Sadesati / サデサティ**：7.5年間）に注意
- ダシャーとゴーチャルが一致した時期に最も強く結果が現れる
