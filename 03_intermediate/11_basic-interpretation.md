# 基本チャート解釈 — Basic Chart Interpretation

> **難易度**: 中級  
> **前提ファイル**: 全ての前ノート（01〜10）  
> **キーワード**: Raja Yoga, Dhana Yoga, Yoga, Interpretation Framework

## 1. チャート解釈とは

チャート解釈とは、出生ホロスコープの情報を統合して、その人の気質・ライフテーマ・人生の流れを読み取る技術です。

初心者が陥りがちな失敗は：
- 一つの惑星や星座だけを見て判断する（断片的な解釈）
- 「この位置は絶対に悪い/良い」と固定的に考える
- ダシャー（時間軸）を無視した解釈

チャートは**複数の要素の複雑な交響曲**です。一つ一つの音符より、全体のハーモニーを聴くことが大切です。

## 2. 7ステップ解釈フレームワーク

### ステップ1：ラグナとラグナ・ロードを確認

**最初に見るべきポイント**

1. ラグナの星座は何か？（元素・様式・支配惑星）
2. ラグナ・ロードはどの室・星座にいるか？
3. ラグナ・ロードの品位は何か？（高揚・自宮・減弱）
4. ラグナ・ロードをアスペクトしている惑星は何か？

> **例**：ラグナが蟹座（Karka）→ ラグナ・ロードは月（Chandra）  
> 月が第10室・山羊座に減弱（Neecha）でいる場合 →  
> 職業（10室）に感情的なアプローチが求められるが、試練を伴う可能性がある。

---

### ステップ2：月の星座とナクシャトラ

**心・感情・精神状態の確認**

1. 月の星座（Chandra Rashi）は何か？
2. 月のナクシャトラは何か？
3. 月の品位・室位置は？
4. 月をアスペクトしている惑星は？

月はその人の**精神世界・感情パターン**を示します。  
月が強い（自宮・高揚・吉星がアスペクト）→ 感情的に安定、直感力が高い  
月が弱い（減弱・凶星がアスペクト）→ 感情の波が大きい、精神的な試練

---

### ステップ3：太陽の星座と室

**自我・活力・人生目的の確認**

1. 太陽の星座と室は？
2. 太陽の品位と近接している惑星（コンバスト）は？

**Combustion（コンバスション / 焼失）**: 惑星が太陽に近すぎる場合（各惑星により度数が異なる）、  
その惑星は「焼失（Combust）」し、弱まります。

| 惑星 | コンバスト度数 |
|------|-------------|
| 月 | 12°以内 |
| 火星 | 17°以内 |
| 水星 | 14°以内（逆行時は12°） |
| 木星 | 11°以内 |
| 金星 | 10°以内（逆行時は8°） |
| 土星 | 15°以内 |

---

### ステップ4：各室の状態を確認

**ライフドメイン別の分析**

重要な室を順に確認します：

| 室 | テーマ | 確認事項 |
|----|-------|---------|
| 1室 | 自己・健康 | 在室惑星、ラグナ・ロードの位置 |
| 2室 | 財産・家族 | バーヴァ・ロード、在室惑星の品位 |
| 4室 | 家庭・母 | バーヴァ・ロード、月との関係 |
| 5室 | 知性・子供 | バーヴァ・ロード、木星との関係 |
| 7室 | 結婚・パートナー | バーヴァ・ロード、金星との関係 |
| 9室 | 師・精神・幸運 | バーヴァ・ロード、木星の影響 |
| 10室 | 職業・社会的地位 | バーヴァ・ロード、土星・太陽の影響 |

---

### ステップ5：惑星の品位と強さを評価

**チャートの「強い惑星」と「弱い惑星」を特定**

強い惑星のテーマは豊かに展開し、弱い惑星のテーマには課題が伴います。

**強い惑星の条件（複数該当するほど強い）**:
- 高揚（Ucha）またはムーラトリコナ
- 自宮（Svagraha）
- ケンドラかトリコナにいる
- 吉星のアスペクトを受けている
- 逆行（一部の観点では強い）

**弱い惑星の条件**:
- 減弱（Neecha）
- ドゥスターナ（6・8・12室）にいる
- 凶星のアスペクトのみを受けている
- コンバスト（太陽に焼失）している

---

### ステップ6：アスペクトとヨーガを確認

**特殊な惑星の組み合わせ（Yoga / ヨーガ）**

**Yoga（ヨーガ）** とは、複数の惑星や室の組み合わせによって生まれる特殊な効果です。  
代表的なヨーガを確認します。

#### Raja Yoga（ラジャ・ヨーガ / 王のヨーガ）

ケンドラ（1,4,7,10室）の支配星とトリコナ（1,5,9室）の支配星が**合・交換・相互アスペクト**する場合に成立。  
社会的成功・権威・名声をもたらします。

**例**：ラグナが蟹座（Karka）の場合
- ケンドラ支配星：月（1室）・金星（4室）・土星（7室）・火星（10室）
- トリコナ支配星：月（1室）・火星（5室）・木星（9室）
- 火星は5室（トリコナ）と10室（ケンドラ）の両方を支配 → **ヨーガ・カラカ**（最強の吉星）

#### Dhana Yoga（ダナ・ヨーガ / 富のヨーガ）

2室・11室の支配星がラグナ・ロードや5室・9室の支配星と関係する場合。  
物質的豊かさ・富の蓄積をもたらします。

#### Gajakesari Yoga（ガジャケーサリ・ヨーガ）

月と木星がケンドラ位置関係（お互いに1・4・7・10室）にある場合。  
知恵・名声・徳のある人生をもたらします。

#### Pancha Mahapurusha Yoga（パンチャ・マハープルシャ・ヨーガ）

5つの惑星（火星・水星・木星・金星・土星）がそれぞれ**ケンドラで自宮または高揚**にある場合。  
それぞれに応じた偉大な人物の素質を示します。

| 惑星 | ヨーガ名 | 特質 |
|------|---------|------|
| 火星 | Ruchaka Yoga | 軍事的才能・リーダーシップ |
| 水星 | Bhadra Yoga | 知性・弁論・ビジネス |
| 木星 | Hamsa Yoga | 知恵・精神性・哲学 |
| 金星 | Malavya Yoga | 芸術・美・物質的繁栄 |
| 土星 | Shasha Yoga | 政治・権力・大衆への影響力 |

---

### ステップ7：現在のダシャーを確認

**時間軸の読み取り**

1. 現在はどのマハーダシャーか？
2. どのアンタルダシャーか？
3. そのダシャー惑星はチャート内でどの室・星座にあり、品位は何か？
4. 現在のゴーチャル（トランジット）との関係は？

ダシャー惑星がチャートで強ければ → その期間に良い結果が出やすい  
ダシャー惑星がチャートで弱ければ → その期間に試練が多くなりやすい

## 3. 解釈の実践ポイント

### 「良い悪い」で考えない

Jyotishの解釈は「良い/悪い」の二分法ではなく、**エネルギーの方向性と可能性**を見ることです。

- 減弱惑星 → 「そのテーマに試練がある」→「その試練を通じて深い理解を得る可能性がある」
- ドゥスターナ（6・8・12室）の惑星 → 「困難がある」→「その困難を通じて成長する」

### 全体バランスを見る

一つの「良くない」要素があっても、他の要素でバランスが取れていることが多いです。  
チャートを「全体として」見る習慣をつけましょう。

### 時代・文化背景を考慮する

古典的なJyotishの解釈（例：7室・金星 = 結婚）は現代的な文脈に合わせて柔軟に解釈します。  
「結婚」→「深いパートナーシップ・関係性」のテーマとして読み替えることも有効です。

---

## 4. 初心者向け：まず確認すべき3つのポイント

複雑なチャートを前にしたとき、まずはこの3つから始めましょう：

1. **ラグナとラグナ・ロード** → その人の本質的な方向性
2. **月の星座とナクシャトラ** → 感情・心の働き
3. **現在のダシャー惑星** → 今の時期のテーマ

これだけでも、基本的な読みが可能です。
