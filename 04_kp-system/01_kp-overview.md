## 1. KP占星術とは

**Krishnamurti Paddhati（クリシュナムルティ・パッダティ / KP占星術）** は、  
インドの占星術師 **Prof. K.S. Krishnamurti（K.S.クリシュナムルティ教授、1908〜1972）** が開発した  
現代的・科学的な占星術システムです。

（*paddhati* = 方法・システム・流儀）

### 開発の背景

クリシュナムルティ教授は伝統的なJyotishを深く学んだ後、以下の課題を感じました：

- 伝統的なJyotishの予測精度が「いつ？」という時期特定に弱い
- 同じチャートでも占星術師によって解釈が大きく異なる
- より**客観的・数学的な根拠に基づく予測**が必要

そこで彼は20年以上にわたる観察・研究から、**ナクシャトラのサブ区分（Sub-Lord）** を使った  
新しい予測システムを確立しました。

### KPの主な特徴

| 特徴 | 伝統的Jyotish | KP占星術 |
|------|------------|---------|
| ハウスシステム | 全宮式（Whole Sign）が主流 | **Placidus（プラシーダス）** |
| アヤナムシャ | Lahiri等、複数 | **KPアヤナムシャ**（独自） |
| 最小単位 | ナクシャトラ（13°20'） | **サブ（Sub）**（最小0°1'46"程度） |
| 予測の焦点 | テーマ・性質・流れ | **具体的な時期・出来事** |
| 客観性 | 解釈の幅が広い | ルールが明確で再現性が高い |

## 3. KPの根本概念：サブロード理論

KPの核心は **Sub-Lord（サブロード）** の概念にあります。

### 3層構造

Jyotishのナクシャトラ（27宿）を、さらにヴィムショッタリー・ダシャーの比率で細分化します：

```
第1層: Star Lord（スターロード / ナクシャトラ・ロード）
         ↓ ナクシャトラ内をさらに9分割
第2層: Sub-Lord（サブロード）
         ↓ Sub内をさらに9分割
第3層: Sub-Sub-Lord（サブサブロード）
```

### なぜナクシャトラをさらに分割するのか

伝統的なJyotishでは「ナクシャトラ・ロードが同じ2人の人」は同じ結果になるはずですが、  
実際には全く異なる人生を歩むことがあります。

KPはその差異が「同じナクシャトラ内のどのサブ（Sub）にいるか」で決まると考えます。

## 4. Sub-Lordの計算原理

### ナクシャトラ内のサブ区分

各ナクシャトラ（13°20' = 800分）をヴィムショッタリーの年数比率で分割：

| 惑星 | ダシャー年数 | サブの割合 | サブの度数幅 |
|------|-----------|----------|------------|
| ケートゥ（Ketu） | 7年 | 7/120 | 0°46'40" |
| 金星（Venus） | 20年 | 20/120 | 2°13'20" |
| 太陽（Sun） | 6年 | 6/120 | 0°40'00" |
| 月（Moon） | 10年 | 10/120 | 1°6'40" |
| 火星（Mars） | 7年 | 7/120 | 0°46'40" |
| ラーフ（Rahu） | 18年 | 18/120 | 2°0'0" |
| 木星（Jupiter） | 16年 | 16/120 | 1°46'40" |
| 土星（Saturn） | 19年 | 19/120 | 2°6'40" |
| 水星（Mercury） | 17年 | 17/120 | 1°53'20" |
| **合計** | **120年** | **120/120** | **13°20'** |

### 27ナクシャトラ × 9サブ = 243サブ区分

黄道全体（360°）は243のサブ区分に分割されます。  
各サブ区分には「スターロード + サブロード」の2惑星が対応します。

**例：牡羊座0°0'0"〜（Ashwiniの始まり）**

| 度数範囲 | スターロード | サブロード |
|---------|-----------|----------|
| 牡羊0°0'0" 〜 0°46'40" | ケートゥ | ケートゥ |
| 牡羊0°46'40" 〜 3°0'0" | ケートゥ | 金星 |
| 牡羊3°0'0" 〜 3°40'0" | ケートゥ | 太陽 |
| 牡羊3°40'0" 〜 4°46'40" | ケートゥ | 月 |
| 牡羊4°46'40" 〜 5°33'20" | ケートゥ | 火星 |
| 牡羊5°33'20" 〜 7°33'20" | ケートゥ | ラーフ |
| 牡羊7°33'20" 〜 9°20'0" | ケートゥ | 木星 |
| 牡羊9°20'0" 〜 11°26'40" | ケートゥ | 土星 |
| 牡羊11°26'40" 〜 13°20'0" | ケートゥ | 水星 |
| 牡羊13°20'0" 〜 ... | 金星 | 金星 |

> **KP表（Sub-Lord Table）**: 243行からなるこの表はKPの根幹です。  
> 惑星の度数さえ分かれば、スターロードとサブロードが一意に決定されます。

## 5. KPのハウスシステム：Placidus

### なぜPlacidusなのか

伝統的なJyotishは**全宮式（Whole Sign）**を使います（各星座 = 1ハウス）。  
KPは**Placidus（プラシーダス）**を使い、ハウスのカスプ（境界点）が星座の途中に来ます。

Placidusは**出生地の緯度・経度と時刻**に基づいて各ハウスのカスプを計算します。  
そのため同じ時刻に生まれても、異なる場所では異なるカスプになります。

### KPでのハウスの見方

KPでは各ハウスの**カスプ（Cusp / 境界点）**の度数が重要です。  
そのカスプがどのナクシャトラのどのサブにあるか = そのカスプの「スターロード」「サブロード」

**カスパル・サブロード（Cuspal Sub-Lord）** = そのハウスのテーマが実現するかを決める最重要ポイント

## 6. KPの核心：カスパル・サブロード理論

### 基本原則

**あるハウスのテーマが実現するかどうかは、そのハウスのカスプのサブロードが決定する。**

カスプのサブロードが「そのハウスの意味を支持する（promise）」場合のみ、  
そのハウスのテーマが人生で実現します。

### 実例：第7ハウス（結婚）

第7ハウスのカスプのサブロードが：
- **第7室・第2室（家族）・第11室（願望成就）** のシグニフィケーターであれば → 結婚する
- **第6室（制限）・第10室（職業優先）・第12室（損失）** のシグニフィケーターであれば → 結婚しない or 大きな遅延

### 実例：第5ハウス（子供）

第5ハウスのカスプのサブロードが：
- **第5室・第2室（家族）・第11室** のシグニフィケーターであれば → 子供を持つ
- **第1室・第4室（孤独な養育）・第12室（喪失）** のシグニフィケーターであれば → 子供を持ちにくい

## 7. シグニフィケーター（Significator）の概念

**Significator（シグニフィケーター / 示唆惑星）** は、あるハウスのテーマを「代表・担当する」惑星です。

### シグニフィケーターの4グループ（重要度順）

KPでは惑星がハウスに「関係する度合い」を4段階で分類します：

| グループ | 関係の内容 | 重要度 |
|---------|----------|--------|
| **Group A** | そのハウスに在室する惑星のナクシャトラにいる惑星 | 最強 |
| **Group B** | そのハウスに在室する惑星 | 強 |
| **Group C** | そのハウスのロード（支配星）のナクシャトラにいる惑星 | 中 |
| **Group D** | そのハウスのロード（支配星）自身 | 弱 |

> **ナクシャトラのスターロード = その惑星のエネルギーを「前面に出す」ポータル**  
> という考え方がKPの核心です。

### シグニフィケーター特定の手順

例：第10ハウス（職業）のシグニフィケーターを調べる場合

1. 第10ハウスに在室する惑星を確認（例：火星、木星）
2. 火星のナクシャトラにいる惑星 → Group A
3. 木星のナクシャトラにいる惑星 → Group A
4. 第10ハウスのロード（例：土星）のナクシャトラにいる惑星 → Group C
5. 土星自身 → Group D

## 8. ルーリング・プラネット（Ruling Planets / RPs）

**Ruling Planets（ルーリング・プラネット / RP）** は、  
判断を行う「その瞬間」を支配している惑星群です。

KPでは「占星術師が判断を行う瞬間にも宇宙的な意味がある」と考えます。

### RPの構成要素（判断時刻における）

| 要素 | 方法 |
|------|------|
| **Lagna Lord（ラグナ・ロード）** | 判断時刻のアセンダントの支配惑星 |
| **Lagna Star Lord** | 判断時刻のアセンダントのナクシャトラ支配惑星 |
| **Moon Sign Lord** | 判断時刻の月の星座支配惑星 |
| **Moon Star Lord** | 判断時刻の月のナクシャトラ支配惑星 |
| **Day Lord** | 判断を行う曜日の支配惑星 |

> 月曜=月、火曜=火星、水曜=水星、木曜=木星、金曜=金星、土曜=土星、日曜=太陽

### RPの使い方

1. **確認**: 予測されるシグニフィケーターとRPが一致しているか？
2. **調整**: 質問の答えが「Yes」の場合、RPの惑星がシグニフィケーターの中にいるはず
3. **時期特定**: RPと一致するダシャー期間が、そのイベントの発生時期

## 9. KPによる時期特定の方法

これがKPの最も強力な技法です。

### ステップ1：ハウスを特定

予測したいテーマに関連するハウスを特定します。

| 予測テーマ | 関連ハウス |
|-----------|---------|
| 結婚 | 2・7・11室 |
| 職業開始 | 2・6・10室 |
| 子供 | 2・5・11室 |
| 海外移住 | 3・9・12室 |
| 病気 | 1・6・8・12室 |
| 住居変更 | 4・11室 |

### ステップ2：カスパル・サブロードを確認

そのハウスのカスプのサブロードが関連ハウスのシグニフィケーターであることを確認。  
（「Promise / 約束」があるか）

### ステップ3：ダシャー・アンタルダシャーを特定

現在のダシャー・アンタルダシャーの惑星が、  
関連ハウスのシグニフィケーターになっているダシャー期間を特定します。

### ステップ4：ゴーチャルで絞り込む

特定のダシャー期間内で、「いつ」を絞り込むためにゴーチャル（トランジット）を使います。

特に **KPゴーチャル** では：
- **木星と土星のトランジット**でおおよその年を特定
- **太陽のトランジット**で月を特定
- **月のトランジット**で日にちを特定

## 10. KPアヤナムシャ

**KPアヤナムシャ（Krishnamurti Ayanamsha）** は、Lahiriアヤナムシャとは若干異なります。

| アヤナムシャ | 2000年1月1日の値 | 特徴 |
|-----------|---------------|------|
| Lahiri | 約23°51'11" | インド政府公式 |
| KP | 約23°51'00" | Lahiriより約11秒小さい |

差異は小さいように見えますが、度数に敏感なKPのサブロード計算では、  
この数十秒の差が「どのサブに入るか」を変える可能性があります。

> **注意**: KPソフトウェアを使う場合は必ず「KP Ayanamsha」を選択してください。  
> 「Lahiri」を選択すると計算結果が微妙にずれます。
