## 1. KP表の読み方

KP表では各惑星の度数に対して：
- **Star Lord（スターロード）**: そのナクシャトラの支配惑星
- **Sub-Lord（サブロード）**: そのサブ区分の支配惑星

が一意に決まります。

## 3. サブロード表の計算式

残りの星座（乙女〜魚）は同じパターンで繰り返します。  
各ナクシャトラの開始惑星は以下の順序：

```
Nak 1  (Ashwini)  → Ke
Nak 2  (Bharani)  → Ve
Nak 3  (Krittika) → Su
Nak 4  (Rohini)   → Mo
Nak 5  (Mrigashira) → Ma
Nak 6  (Ardra)    → Ra
Nak 7  (Punarvasu) → Ju
Nak 8  (Pushya)   → Sa
Nak 9  (Ashlesha) → Me
Nak 10 (Magha)    → Ke（繰り返し）
...
```

サブの開始惑星は「そのナクシャトラのスターロードと同じ惑星から始まる」ルールです。

### 度数の計算式

```
サブの幅（度数）= ナクシャトラの度数幅（13°20'） × （惑星のダシャー年数 / 120）
```

| 惑星 | ダシャー年 | サブの幅 |
|------|---------|---------|
| Ke | 7 | 0°46'40" |
| Ve | 20 | 2°13'20" |
| Su | 6 | 0°40'0" |
| Mo | 10 | 1°6'40" |
| Ma | 7 | 0°46'40" |
| Ra | 18 | 2°0'0" |
| Ju | 16 | 1°46'40" |
| Sa | 19 | 2°6'40" |
| Me | 17 | 1°53'20" |

## 4. 実践：惑星のスターロードとサブロードを求める

### 例：火星が乙女座15°30'にある場合

1. 乙女座15°30' はどのナクシャトラか？
   - 乙女10°0'〜23°20' = Hasta（ハスタ）, スターロード = **Mo（月）**

2. Hastaの中で15°30'はどのサブか？
   - Hasta = 乙女10°0'〜23°20'（スターロード：Mo）
   - サブの計算：

| 開始 | 終了 | サブロード |
|------|------|----------|
| 10°0'0" | 11°6'40" | Mo |
| 11°6'40" | 11°53'20" | Ma |
| 11°53'20" | 13°53'20" | Ra |
| 13°53'20" | 15°40'0" | Ju |
| 15°40'0" | 17°46'40" | Sa |
| ... | ... | ... |

   - 15°30'は「13°53'20"〜15°40'0"」の範囲 → サブロード = **Ju（木星）**

**結論：火星のスターロード = 月（Mo）、サブロード = 木星（Ju）**

この場合、火星は「月の星座にある惑星のテーマ」を前面に出し、  
「木星のシグニファイする事柄」という色をつけて表現します。

## 5. KP表自動計算のためのアルゴリズム

プログラムでKPの度数を計算したい場合の擬似コード：

```python
# サブロード表の計算
dasha_years = [7, 20, 6, 10, 7, 18, 16, 19, 17]  # Ke, Ve, Su, Mo, Ma, Ra, Ju, Sa, Me
dasha_planets = ['Ke', 'Ve', 'Su', 'Mo', 'Ma', 'Ra', 'Ju', 'Sa', 'Me']
nakshatra_span = 13 + 20/60  # 13°20' = 13.333...°

def get_star_lord(degree):
    """0〜360°の度数からスターロードを返す"""
    nakshatra_num = int(degree / nakshatra_span)  # 0〜26
    star_lord_index = nakshatra_num % 9
    # Ashwiniの開始はKe（index 0）から
    return dasha_planets[star_lord_index]

def get_sub_lord(degree):
    """0〜360°の度数からサブロードを返す"""
    nakshatra_num = int(degree / nakshatra_span)
    start_in_nakshatra = degree - nakshatra_num * nakshatra_span
    
    # このナクシャトラのスターロードから始まるサブ
    star_lord_index = nakshatra_num % 9
    
    # サブの累積度数を計算
    cumulative = 0
    for i in range(9):
        sub_index = (star_lord_index + i) % 9
        sub_span = nakshatra_span * dasha_years[sub_index] / 120
        cumulative += sub_span
        if start_in_nakshatra < cumulative:
            return dasha_planets[sub_index]
```
