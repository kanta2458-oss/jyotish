## 1. 予測の全体フロー

KPでの予測は以下のフローで進みます：

```
1. 質問・テーマに関連するハウスを特定
        ↓
2. カスパル・サブロードを確認（Promise があるか）
        ↓
3. シグニフィケーターを特定（誰が担当するか）
        ↓
4. ルーリング・プラネット（RP）との一致確認
        ↓
5. ダシャー・アンタルダシャーで時期を絞り込む
        ↓
6. ゴーチャル（トランジット）で日時を確定
```

## 2. ハウスの組み合わせ（テーマ別）

各イベントには「支持するハウス」と「否定するハウス」があります。

### 主要イベントのハウス組み合わせ

| イベント | 支持するハウス | 否定するハウス |
|---------|-------------|-------------|
| **結婚** | 2・7・11 | 1・6・10・12 |
| **子供の誕生** | 2・5・11 | 1・4・10・12 |
| **職業・就職** | 2・6・10 | 1・5・8・12 |
| **海外移住・渡航** | 3・9・12 | 1・4 |
| **家・不動産の購入** | 4・11・12 | 1・5・8 |
| **財産・収入増加** | 2・6・10・11 | 1・8・12 |
| **病気** | 1・6・8・12 | 1・5（回復） |
| **死別・大きな喪失** | 2・7・8 | 1・5 |
| **精神的成長・学習** | 4・5・9・12 | — |

> **なぜ2室と11室がほぼ常に含まれるのか？**  
> 2室 = 家族・物の追加（蓄積）  
> 11室 = 願望の成就・利益  
> どのポジティブなイベントも「家族/蓄積 + 願望成就」を伴うというKPの基本原則

## 3. カスパル・サブロードの詳細分析

### Promise（約束）の確認方法

例：**第7ハウスカスプが乙女座12°に来る場合の結婚予測**

**手順1**: 乙女座12°のスターロードとサブロードを確認

乙女座10°〜23°20' = Hasta（スターロード：月）  
乙女12°のサブを表で確認 → 例として**サブロード：木星（Ju）**とする

**手順2**: 木星（サブロード）が結婚関連ハウス（2・7・11室）のシグニフィケーターかを確認

- 木星が第7室の在室惑星のナクシャトラにいる → Group A
- 木星が第7室にいる → Group B
- 木星が第7室ロードのナクシャトラにいる → Group C
- 木星が第7室ロード → Group D

いずれかで木星が2・7・11室のシグニフィケーターであれば → **Promise あり（結婚する）**

木星が1・6・10・12室のシグニフィケーターのみであれば → **Promise なし（結婚しにくい）**

### 複雑なケース：サブロードが否定室と肯定室の両方に関係する場合

例：サブロードが7室（肯定）かつ6室（否定）のシグニフィケーターの場合

→ 結婚はするが、離婚・別居・関係の問題が生じやすい  
→ 肯定室の数が否定室より多ければ、総合的にはポジティブと判断

## 4. シグニフィケーターの特定：詳細手順

### 例：第10ハウス（職業）のシグニフィケーターを全て特定する

**前提チャートの設定（架空の例）**:
- ラグナ：蟹座（Karka）
- 第10ハウスの星座：牡羊座（Mesha）→ 第10ハウスロード = 火星（Ma）
- 第10ハウスに在室：木星（Ju）
- 木星のナクシャトラ：Uttara Ashadha（スターロード：太陽 Su）
- 火星のナクシャトラ：Ashlesha（スターロード：水星 Me）

**Group A（最強）**: 第10ハウスの在室惑星（木星）のナクシャトラに位置する惑星
- Uttara Ashadhaにいる惑星を全て調べる
- 例：土星がUttara Ashadhaにいるとすれば → **土星はGroup A**

**Group B（強）**: 第10ハウスに在室する惑星
- 木星が在室 → **木星はGroup B**

**Group C（中）**: 第10ハウスロード（火星）のナクシャトラにいる惑星
- AshleshaにいるすべてのチャートのChandra（月）が→ **月はGroup C**

**Group D（弱）**: 第10ハウスロード（火星）自身
- **火星はGroup D**

**最終シグニフィケーター一覧（例）**:
土星（A）、木星（B）、月（C）、火星（D）

## 5. ダシャーによる時期特定

### ステップ1：現在のダシャー・アンタルダシャーを確認

例：現在が**ラーフ・マハーダシャー / 木星・アンタルダシャー**とする

### ステップ2：ダシャー惑星がシグニフィケーターかを確認

- ラーフ → 第10ハウスのシグニフィケーターか？（上記のGroup A〜Dにあるか）
- 木星 → 第10ハウスのシグニフィケーターか？

木星がGroup Bにあるので → **木星アンタルダシャーの期間は職業イベントが起きやすい**

### ステップ3：プラティヤンタルダシャーで月を絞り込む

ラーフ・マハーダシャー / 木星・アンタルダシャー内で、  
さらに第10ハウスのシグニフィケーターのプラティヤンタルダシャー期間が対象時期

## 6. ルーリング・プラネット（RP）の実践

### 判断時刻のRPを計算する

例：2026年4月4日（土曜日）、午後3時（東京）に判断する場合

| 要素 | 計算 | 惑星 |
|------|------|------|
| Day Lord | 土曜日 | **土星（Sa）** |
| Moon Sign Lord | 月が双子座にいるとして | **水星（Me）** |
| Moon Star Lord | 月がArdra（アールドラー）にいるとして | **ラーフ（Ra）** |
| Lagna Sign Lord | 午後3時の東京のラグナが蠍座として | **火星（Ma）** |
| Lagna Star Lord | 蠍座15°のナクシャトラはAnuradha（アヌラーダー）→ スターロード | **土星（Sa）** |

この場合のRP：**土星、水星、ラーフ、火星**

### RPとシグニフィケーターの一致を確認

シグニフィケーター（例：土星・木星・月・火星）とRP（土星・水星・ラーフ・火星）の共通項：

**土星と火星** → この2惑星がRPとシグニフィケーターの両方に含まれている

→ 判断：現在の時期は職業関連のイベントが起きやすいタイミング

## 7. Prashna（プラシュナ / 質問占星術）への応用

**Prashna KP（プラシュナ KP）** は、出生チャートではなく  
「質問した瞬間のチャート」を使って判断するKPの応用です。

伝統的なJyotishのプラシュナ（Horary Astrology）をKPシステムで行います。

### Prashna KPの特徴

1. 出生時刻が不明でも使える
2. 「今この質問に答えは出るか」を即座に判断できる
3. 占星術師が計算する瞬間のチャートを使う

### 手順

1. 質問を受けた瞬間の時刻・場所でKPチャートを作成
2. 質問テーマに関するハウスを特定
3. カスパル・サブロードの Promise を確認
4. シグニフィケーターとRPを照合
5. 現在のダシャーで時期を判断

> 例：「今の仕事を続けるべきか？」  
> → 第6室（現在の仕事）と第10室（職業）のカスパル・サブロードが  
> 第6・10室を示唆 → 仕事継続を支持

## 8. 実践的な注意点

### KP特有の解釈原則

1. **否定室が多いとき**  
   「そのテーマは実現しない」ではなく「時期の遅延・困難」と解釈することも多い

2. **サブロードが複数ハウスに関係する**  
   主要ハウス（7室等）の関与が最低1室は必要。その上で否定室との兼ね合いを考える

3. **ラーフ・ケートゥのシグニフィケーター**  
   ラーフとケートゥは自身が在室する星座のロードの代役を務めることがある  
   また、コンジャンクション（合）している惑星のエネルギーを引き継ぐ

4. **惑星の委任（Delegation）**  
   Group Aの惑星が最強とされるが、実際には「チェーン」が形成される場合がある  
   例：火星→火星のナクシャトラに太陽→太陽のナクシャトラに月 = 月が最終的な担当
