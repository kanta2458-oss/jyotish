## 1. KP計算ソフトウェアの必要性

KPの計算（サブロード・Placidusカスプ・KPアヤナムシャ）は手計算で行うことも可能ですが、  
非常に煩雑なため**ソフトウェアの使用が実質的に必須**です。

---

## 2. 主要なKP対応ソフトウェア

### Jagannatha Hora（ジャガンナータ・ホーラ）

- **価格**: 無料
- **プラットフォーム**: Windows（Wineを使えばMac/Linux可）
- **KP機能**: あり（KPアヤナムシャ・Placidus・Sub-Lord表示対応）
- **入手**: 公式サイト（Shyamasundara Dasa氏作成）
- **特徴**: 伝統的Jyotishも含む多機能。KP専用ではないがKPも対応

**設定注意点**:
1. アヤナムシャを「KP (Krishnamurti)」に設定
2. ハウスシステムを「Placidus」に設定
3. KPチャート画面でStar Lord・Sub Lordが表示されることを確認

---

### AstroVision KP Software

- **価格**: 有料（インドのソフトウェア）
- **プラットフォーム**: Windows
- **KP機能**: 非常に充実（KP専用設計）
- **特徴**: KPに特化した最も使いやすいソフトのひとつ。Sub-Lord表・シグニフィケーター計算・RP計算を自動化

---

### KP Astrologer（モバイル）

- **価格**: 一部無料・有料版あり
- **プラットフォーム**: Android / iOS
- **特徴**: スマートフォンでKPチャート確認が可能。外出先での参照に便利

---

### Shri Jyoti Star

- **価格**: 有料（高機能・高価格帯）
- **プラットフォーム**: Windows / Mac
- **特徴**: 伝統的Jyotish・KP・Jaimini等を統合。研究用途に最適

---

## 3. Jagannatha Hora でのKP設定手順

### インストール後の初期設定

1. **Tools → Preferences** を開く

2. **Ayanamsha（アヤナムシャ）** の設定:
   ```
   Options → Ayanamsha → Krishnamurti (KP)
   ```

3. **House System（ハウスシステム）** の設定:
   ```
   Options → House Division → Placidus
   ```

4. **KP Chart表示**:
   ```
   Charts → KP Chart
   ```
   ここでStar LordとSub Lordが表示されます。

### KPチャート画面の読み方

KPチャート画面では各惑星について以下が表示されます：

```
Planet | Sign | Degree | Nakshatra | Star Lord | Sub Lord
-------+------+--------+-----------+-----------+---------
Sun    | Ar   | 14°30' | Bharani   | Venus     | Jupiter
Moon   | Ge   |  8°15' | Ardra     | Rahu      | Mars
...
```

各ハウスのカスプについても同様に表示：

```
Cusp | Sign | Degree | Star Lord | Sub Lord
-----+------+--------+-----------+---------
1    | Ca   | 22°15' | Moon      | Saturn
2    | Le   |  8°30' | Sun       | Jupiter
...
```

## 5. KP計算のオンラインツール

ソフトウェアをインストールできない場合のオンライン代替：

- **astro.com** → Extended Chart SelectionでayanamshaをKrishnamurtiに設定可能
  - ただし完全なKP機能はなく、参考用

- **Kp-System.com** 等のKP専門サイト → チャート生成機能あり（英語）

## 6. KP計算の手動確認方法

ソフトウェアの結果を検証したい場合の手計算手順：

### 1. サイデリアル度数の計算

```
サイデリアル度数 = トロピカル度数 - KPアヤナムシャ
```

2026年のKPアヤナムシャ ≒ 24°1'（概算、毎年約50.2"増加）

```
例：太陽がトロピカルで牡羊座15°の場合
サイデリアル（KP）= 牡羊15° - 24°1' = 魚座20°59'
```

### 2. ナクシャトラの特定

```
サイデリアル度数 ÷ 13.333...° = ナクシャトラ番号（0〜26）
```

魚座20°59' = 330°59'（0°牡羊からの通算）
330°59' ÷ 13.333... ≒ ナクシャトラ番号 24（0始まり）= 25番目 = **Purva Bhadrapada**

### 3. サブの特定

サブロード表（前のノート参照）を使い、  
ナクシャトラ内の位置（残余度数）からサブロードを特定
